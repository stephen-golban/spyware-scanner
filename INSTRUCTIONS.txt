==============================================================================
 HOW TO DOWNLOAD AND RUN THE SURVEILLANCE TOOLKIT  (Windows 10 / 11)
 Downloading from GitHub - step by step
==============================================================================

 This toolkit has TWO tools:

   1. THE SCAN  - only looks for spying (screen recording, camera/mic spying,
                  keyloggers, remote-access tools...). Changes nothing.
   2. THE FIX   - optional. Turns off risky settings and removes flagged
                  programs. Asks before every change, and everything is
                  reversible.

 Do the SCAN first (Parts 1-3). Only do the FIX (Part 4) if you need to and you
 understand what you are changing.

 Files you will see after downloading (double-click the numbered .bat files;
 the "engine" .ps1 files run automatically - you never click them directly):

     1_SCAN.bat            <- the scan        (double-click this first)
     2_FIX.bat             <- the fix         (optional, after the scan)
     3_UNDO.bat            <- undo the fix

     scan-engine.ps1       <- used by 1_SCAN.bat   (don't click directly)
     fix-engine.ps1        <- used by 2_FIX/3_UNDO (don't click directly)
     README.md             INSTRUCTIONS.txt   <- docs


==============================================================================
 PART 1 - DOWNLOAD THE FILES FROM GITHUB
==============================================================================

   1. On the PC, open a web browser (Edge or Chrome) and go to the repository:

          (paste your GitHub repository link here)

   2. Click the green button near the top-right that says   " < > Code ".

   3. In the menu that drops down, click   "Download ZIP".

   4. The browser may warn that the ZIP "could harm your device" (it says that
      about anything containing scripts). Choose  "Keep"  /  "Keep anyway".

   5. Open your Downloads folder (click the download arrow in the browser, or
      open File Explorer and click "Downloads" on the left).

   6. RIGHT-click the ZIP you just downloaded and choose  "Extract All..."  then
      click  "Extract".

   7. A folder opens. You may need to open ONE more folder inside it (GitHub
      names it something like "...-main"). Keep opening until you can see
      1_SCAN.bat and the other files listed above sitting together.

      >> Keep all the files together in this one folder. The launchers look for
         the scripts next to them.

   (Optional, only if a script later refuses to run: right-click
    1_SCAN.bat -> Properties -> if you see an "Unblock" checkbox near the
    bottom, tick it and click OK. Do the same for the .ps1 files. Windows marks
    downloaded files as "blocked" and this clears it.)


==============================================================================
 PART 2 - RUN THE SCAN
==============================================================================

   1. DOUBLE-CLICK the file named:   1_SCAN.bat

   2. You will probably see a blue box: "Windows protected your PC".
      This is normal for a downloaded script. To continue:
          - Click the small text  "More info"
          - Then click the button  "Run anyway"

   3. A window pops up asking:
          "Do you want to allow this app to make changes to your device?"
      Click  "Yes".

      >> Why: this only lets the scan SEE the whole computer (all accounts,
         services, tasks). It still changes nothing.

   4. A black window opens and text scrolls by. LET IT RUN - about 1 to 2
      minutes.

   5. When it finishes, a report opens automatically in your browser. A copy is
      also saved to the DESKTOP, named:
          Surveillance_Scan_<computer-name>_<date>.html   (the nice report)
          Surveillance_Scan_<computer-name>_<date>.txt    (a plain-text copy)

   6. Press the ENTER key to close the black window.


==============================================================================
 PART 3 - READ THE SCAN REPORT
==============================================================================

   Findings are sorted into three levels:

     HIGH    - Look first. E.g. something is using the camera right now, or a
               known spying program was found.
     MEDIUM  - "Please check." MANY of these are normal (Zoom/Teams using the
               camera; an IT remote-support tool). Read each and decide if it
               is expected.
     INFO    - Background info, usually fine.

   A long list is not automatically bad. Read the HIGH items, then skim the
   rest. If HIGH is empty and the MEDIUM items are things you recognise, the
   machine most likely looks clean.


==============================================================================
 PART 4 - RUN THE FIX TOOL  (optional - only if you need to change something)
==============================================================================

   The fix tool asks Y/N before EVERY change, and everything can be undone
   (Part 5). Run it only after reading the scan report.

   1. DOUBLE-CLICK:   2_FIX.bat

   2. Same as before: if "Windows protected your PC" appears, click
      "More info" -> "Run anyway".  Then click "Yes" on the admin prompt.

   3. It walks you through FOUR stages. For each item it shows what it found and
      waits for you to answer. Type the letter and press ENTER.

      STAGE 1 - HARDENING
        Offers to turn off Remote Assistance, Remote Desktop (RDP), and Xbox
        Game Bar recording. Type  y  to turn something off, or  n  to leave it.
        (Say  n  to Remote Desktop if you or IT actually use it.)

      STAGE 2 - STARTUP ITEMS
        Lists each program that launches automatically. Anything matching a
        known spy/monitor name is shown in RED. Type  y  to disable an entry
        (reversible) or  n  to keep it. Say  n  to things you recognise as
        normal (Steam, Discord, audio drivers, your e-signature software, etc.).

      STAGE 3 - DETECTED PROGRAMS
        For anything matching the spying/monitoring name lists, you get three
        choices - type one letter:
            U = Uninstall using the program's own uninstaller (cleanest)
            Q = Quarantine: stop it and MOVE its files aside (reversible)
            S = Skip / leave it alone
        If you are unsure, choose  S  or  Q  (Q is reversible; a full uninstall
        of the wrong thing is not).

      STAGE 4 - PERMANENT DELETE
        Optional. Shows what is sitting in quarantine. To erase it forever you
        must TYPE the word  DELETE  and press Enter. Anything else keeps it.
        If you are not 100% sure, just press Enter and keep it - you can delete
        later, or restore it.

   4. When it finishes, a log and a "Quarantine" folder are saved on the Desktop
      inside a folder called  Surveillance_Fix .


==============================================================================
 PART 5 - UNDO THE FIX  (if you changed something by mistake)
==============================================================================

   1. DOUBLE-CLICK:   3_UNDO.bat
   2. Click "Yes" on the admin prompt.
   3. It reverses everything from the last fix run - puts settings back,
      re-enables services, and moves quarantined files back where they were.

   The only things it CANNOT undo: a program you chose to fully UNINSTALL, and
   a quarantine you chose to PERMANENTLY DELETE.


==============================================================================
 IF SOMETHING GOES WRONG
==============================================================================

   * Double-clicking does nothing, or opens the file in Notepad:
       RIGHT-click the .bat file and choose  "Run as administrator".

   * The black window flashes and disappears instantly:
       RIGHT-click 1_SCAN.bat and choose "Run as administrator".

   * Antivirus blocks it or puts it in "quarantine":
       This happens because these are unsigned scripts that inspect the system.
       They are safe and (for the scan) read-only. Temporarily allow / restore
       the file in your antivirus, then run 1_SCAN.bat again.

   * "Could not get administrator rights":
       Right-click the .bat and choose "Run as administrator".


==============================================================================
 GOOD TO KNOW
==============================================================================

   - The SCAN is read-only and needs no internet. Nothing is changed or sent.
   - The FIX asks before every change and is reversible (except an explicit
     uninstall or permanent delete).
   - No tool catches EVERY kind of spying. It cannot see a hidden physical
     camera in the room, a device plugged in between the keyboard and PC, or a
     skilled custom implant. See README.md for the full limitations.
   - For confirmed real malware, ALSO run a Microsoft Defender Offline scan
     and/or Malwarebytes - they remove infections more thoroughly than this
     tool. Keep the saved report as evidence; do not rush to delete if a
     specific person might be involved.
   - A reboot after making changes is a good idea.
